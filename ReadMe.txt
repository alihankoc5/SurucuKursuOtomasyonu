---- PROGRAMIN ÇALIŞMASI İÇİN -----

1- XAMPP programı Apache ve MySQL paketleri dahil olacak şekilde kurulacak.
	https://www.apachefriends.org/tr/download.html

2- Tarayıcıdan localhost/phpmyadmin adresine giderek yeni bir veritabanı oluşturulmalı,
Veritabanı adı "proje2db" ve karakter seti utf8_turkish_ci olmak zorundadır.

3- Rar dosyası içerisindeki sql.txt içerisindeki kodlar aynı panelin SQL sekmesi içine
aktarılarak çalıştırılmalıdır.

4- Proje dosyaları içerisinde Libraries klasöründe mysql_connector jar kütüphanesi
eksik ise aşağıdaki bağlantıdan ilgili kütüphane indirilip import edilmeli
	https://dev.mysql.com/get/Downloads/Connector-J/mysql-connector-java-5.1.49.tar.gz
